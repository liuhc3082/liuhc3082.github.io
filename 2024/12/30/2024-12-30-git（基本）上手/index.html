git（基本）上手

发表于 2024-12-30 更新于 2025-01-18 本文字数： 5.5k 阅读时长 ≈ 10 分钟

https://www.bilibili.com/video/BV1HM411377j?spm_id_from=333.788.player.switch&vd_source=750577b3795032575c17c9e1387bd61b&p=5

名称 位置
工作区 working directory .git所在目录 ↓add↓
暂存区 staging area .git/index 临时存储即将提交的内容 ↓commit↓
本能地仓库 local reposito .git/object 存储代码和版本信息的主要位置

git commit 后文件将由已暂存改为未修改

git commit 后进入vim界面

i进入编辑模式

在＃上面输入提交信息

按esc退出，并输入:wq提交

git提示“warning: LF will be replaced by CRLF”

LF是换行\n

CRLF是回车\r和换行\n

Git提示“warning: LF will be replaced by CRLF”最详细解释+解决方案

使用 vi <文件名> 修改文件内容（——vim基本使用方法）

插入 i --insert--

退出 esc

保存：

  • 如果你没有做任何改变，输入 :q，然后按 Enter/return
  • 如果你做了一些改变，并希望保留它们，输入 :wq 并按 Enter/return
  • 如果你做了一些修改，并希望放弃它们，请输入 :q! 并按 Enter/return
